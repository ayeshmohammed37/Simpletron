Simpletron simulator:
  initialize memory[100]
  
  initialize special registers
    accumulator,
    instructionCounter(store number of memory location contining the instruction being performed)
    operationCode(left most two digits),
    operand(store number of memory location of on which current instruction operates right most two digits)
    instructionRegister(store the next instruction to be performed)
  
  print welcomeMsg
  
  load program

  execute program
    instruction execution cycle
      fetch instruction from memory[instructionCounter] and store it in instructionRegister
      extract operationCode and operand from instructionRegiste
      use switch to determine the operation code
        execute operation
        update instructionCounter

  print computerDumpMsg


check for erroes
  loading phase
    each number must be in the range (-9999 to +9999), keep prompting the user to re-enter the number untill a correct number is entered
  exection phase
    fetal errors
      divide by zero
      invalid operation code
      accumulator overflows(arithmetic operation resulting calues out of range +9999 to -9999)
